Install Dependencies
=====================

MacOs
-------------------------

Install `Boost <https://www.boost.org>`_::

   brew install boost

Install `Eigen <https://eigen.tuxfamily.org>`_::

   brew install eigen


Linux
-------------------------

Install `Boost <https://www.boost.org>`_::

   sudo apt update
   sudo apt install -y libboost-all-dev

Install `Eigen <https://eigen.tuxfamily.org>`_::

   sudo apt update
   sudo apt install libeigen3-dev
